Lightweight Secure Data Leakage Detection System

Overview
The **Lightweight Secure Data Leakage Detection System** is designed to protect sensitive data from unauthorized access and leakage in cloud environments.  
This project uses a lightweight security model with controlled access and encryption techniques to ensure data confidentiality and integrity.

Objectives
- Prevent unauthorized data access
- Detect possible data leakage
- Provide secure communication between users
- Maintain data privacy in cloud storage

Features
- User Authentication (Login & Registration)
- Data Owner Module
- Cloud Server Module
- Data User Module
- Trusted Authority Module
- Secure File Upload & Download
- Data Leakage Detection Mechanism

Technologies Used
- **Programming Language:** Java
- **IDE:** NetBeans / Eclipse
- **Database:** MySQL
- **Libraries Used:**
  - JTattoo (UI Design)
  - JFreeChart (Graph Visualization)
 
How to Run the Project

1. Clone the repository:
2. Open the project in **vs code**

3. Configure database:
   - Create MySQL database
   - Update DB credentials in `DBConnection.java`

4. Run the project:
   - Execute `Main.java`

Modules Description

User Authentication
Handles secure login and registration of users.

Data Owner
Uploads files to cloud and manages access permissions.

Cloud Server
Stores encrypted data and handles file requests.

Data User
Requests and downloads files with proper authorization.

Trusted Authority
Verifies users and ensures secure key distribution.

Security Features

- Data Encryption
- Access Control Mechanism
- Secure Key Management
- Leakage Detection Strategy
